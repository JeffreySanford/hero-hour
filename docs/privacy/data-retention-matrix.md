# Data Retention Matrix

| Entity                | Retention Policy                |
|-----------------------|---------------------------------|
| User                  | Until account deletion          |
| AuthSession           | 90 days after last activity     |
| LifeProfile           | User-controlled                 |
| Game Progression      | Until account deletion          |
| Strategy Profile      | User-controlled                 |
| Analytics Events      | 2 years (default)               |
| Reflection/Review     | User-controlled                 |
| Habit Anchors         | User-controlled                 |
| Admin Logs            | 2 years (default)               |
