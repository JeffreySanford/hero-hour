# Animation Reference

## Core Screens

- Onboarding
- Home
- Quest/Focus
- Reflection/Review
- Admin

## Animation Types

- Entry transitions
- Idle animations
- Success animations
- Milestone animations
- Reduced-motion alternatives

## Example Animations

- Onboarding: Animated world reveal, avatar selection bounce, CTA glow
- Home: Village growth, streak flame, companion idle
- Quest: Timer pulse, XP burst, completion confetti
- Reflection: Calm fade-in, supportive feedback
- Admin: Chart transitions, data point highlights

## Reduced-Motion

- All animations have fade or static alternatives for accessibility
