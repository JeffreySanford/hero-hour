# Onboarding Wireframes

## Screens to Wireframe

- Intro screen (animated world, CTA)
- Account creation (email, password, SSO)
- Email verification
- Avatar selection
- Life-shape profile
- Schedule profile
- Priorities/frictions
- Habit anchors
- World reveal
- First plan

## Wireframe Descriptions

### Intro Screen

- Animated twilight world with parallax, fireflies, and a central CTA: "Begin Your Journey"
- Option for "Preview the World" and "I Already Have an Account"

### Account Creation

- Email/username, password, confirm password fields
- SSO buttons (Google/Apple)
- Helper text for privacy and password strength

### Email Verification

- Field for verification code
- Actions: Verify, resend, use different email

### Avatar Selection

- Carousel of archetypes (Builder, Scholar, Guardian, etc.)
- Selection highlights and flavor text

### Life-Shape Profile

- Multi-select options for life roles (employee, student, parent, etc.)
- Helper text: "Choose all that apply"

### Schedule Profile

- Time pickers for wake/sleep/work/commute
- Toggle for weekday/weekend difference

### Priorities/Frictions

- Multi-select for priorities and friction points
- Helper text: "These shape your early quests"

### Habit Anchors

- List of common anchors (coffee, lunch, walk, etc.)
- Add custom anchor option

### World Reveal

- Animated world elements appear based on profile
- Avatar and companion appear

### First Plan

- List of 3 main quests, 2 side quests
- Editable before accepting
