# Program Increment (PI) Roadmap

## PI 1 — Foundation and Core Gameplay Loop

- Identity, onboarding, timer reliability, XP, mode switching, life areas, starter world growth, side quests, offline trust

## PI 2 — Habit Formation and Planning Intelligence Foundations

- Daily planning, estimation, decomposition, streaks, focus blocks, reflections, reviews, reward loops

## PI 3 — Deep Progression and Personal Attachment

- Village expansion, avatar progression, companion systems, achievements, roadmap, challenges, guidance profile

## PI 4 — Admin Analytics, Strategy Intelligence, and Recommendation Expansion

- Telemetry rollups, operator dashboards, strategy snapshots, recommendation engine, analytics
