# Sprint 2.2 — Streaks, Focus Sessions, and Reward Loop Deepening

## Feature Focus

- Consistency systems
- Focus sessions
- Reward scaling
- Re-entry support

## User Stories

- As a user, I want a streak for daily participation so that I am motivated to return.
- As a user, I want focus sessions so that I can work in structured blocks.
- As a user, I want deeper reward effects for meaningful completions so that progress feels bigger.
- As a user, I want welcome-back prompts after absence so that re-entry is easy.

## Technical Tasks

- Implement streak engine and streak snapshot UI
- Build focus block mode and bonus reward rules
- Add re-entry summary API and returning-user prompts
- Expand reward animation hooks and reward-tier logic
- Add streak and focus telemetry events

## Acceptance Criteria

- Daily and focus streaks work end-to-end
- Focus sessions can award bonus rewards
- Returning users receive meaningful next-step prompts

## Estimates / Priority / Dependencies

- Priority: High
- Story Points: 32
- Dependencies: Sprint 2.1 planning structures
