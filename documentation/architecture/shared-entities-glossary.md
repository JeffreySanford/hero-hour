# Shared Entities & Glossary

## Core Entities

- User: canonical account identity
- AuthSession: refresh/session record
- Role: named authorization role
- UserRoleAssignment: scope-aware role grant
- LifeProfile: structured real-life context
- UserGameProfile: player-facing identity and progression seed
- QuestSession: tracked timer session
- DailyQuest: plan item for a day
- SubQuest: decomposed child quest
- HabitAnchor: recurring routine hook
- VillageState: world progression state
- CompanionState: companion progress and mood
- AchievementProgress: progress toward achievements
- ChallengeProgress: progress toward challenges
- DailyReflection: end-of-day check-in
- WeeklyReviewSnapshot: weekly summary aggregate
- UserStrategySnapshot: private user strategy scorecard
- RecommendationOutput: explainable suggestion generated from signals
- AnalyticsEvent: event log used for rollups and admin analytics

## Domain Terms

- Realm / Life Area: major life category like Work, Health, Family, Learning, Rest
- Main Quest: high-priority planned item
- Side Quest: low-pressure optional behavior prompt
- Focus Block: uninterrupted work session
- Overcommitment: planning beyond realistic capacity
- Balance Score: how evenly effort is distributed across chosen life domains
- Strategy Profile: private coaching-oriented interpretation of behavior
