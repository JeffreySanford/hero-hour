# Agile Documentation – Time-Forge: Life Quest

This folder contains all agile planning, PI (Program Increment) overviews, sprint backlogs, and acceptance criteria for the project.

## Structure

- `PI-1.md` – Program Increment 1 plan
- `PI-Backlog.md` – Sprint backlog and user stories for PI 1
- `PI-AcceptanceCriteria.md` – Acceptance criteria for all PIs

## Usage

- Update these files at the start and end of each PI and sprint
- Use as the single source of truth for agile planning and progress tracking
- Add new PI and sprint files as the project evolves
