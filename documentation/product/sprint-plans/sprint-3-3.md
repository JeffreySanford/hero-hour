# Sprint 3.3 — Challenges, Roadmap, and Strategy Profile MVP

## Feature Focus

- Weekly/monthly challenges
- Roadmap
- First personal strategy profile

## User Stories

- As a user, I want weekly and monthly challenges so that I have goals beyond the day.
- As a user, I want a roadmap of future milestones so that progression feels directed.
- As a user, I want a private strategy profile so that I can improve planning and execution.
- As a user, I want behavior-based suggestions so that the app helps me become more effective.

## Technical Tasks

- Create ChallengeDefinition, ChallengeProgress, MilestoneTrack, and UserStrategySnapshot entities
- Build challenge cards, milestone roadmap, and strategy profile screen
- Add first score dimensions: planning, focus, completion, balance
- Add first deterministic rules: overcommitment and imbalance
- Add private strategy-profile API surface

## Acceptance Criteria

- Weekly and monthly challenge progress updates automatically
- Roadmap shows future milestones
- Strategy profile displays private scorecards and suggestions

## Estimates / Priority / Dependencies

- Priority: Medium-High
- Story Points: 38
- Dependencies: PI 2 review and telemetry foundations
