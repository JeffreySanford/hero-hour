# Planning Documentation Overview

This directory contains the expanded, verbose planning artifacts for Time-Forge: Life Quest, including:

- **Epics**: High-level product capabilities and themes for each Program Increment (PI)
- **Features**: SAFe-style features with benefit hypotheses, acceptance criteria, and user stories
- **Stories**: (To be expanded) Sprint-sized implementation stories for each feature
- **Copilot Packs**: Implementation-ready prompts, file trees, and guardrails for Copilot-driven development

## Structure

- `epics/` – PI-level goals and epic breakdowns
- `features/` – SAFe-style features and user stories
- `stories/` – (Planned) Sprint-level user stories and tasks
- `copilot-packs/` – (Planned) Sprint-by-sprint Copilot build packs

## Usage

- Use these documents to drive backlog creation, sprint planning, and Copilot/AI-powered implementation
- Update as new PIs, features, or sprints are defined
- Expand `stories/` and `copilot-packs/` as the project matures
