# Data Handling Workflows

## Deletion/Anonymization

- User-initiated deletion removes or anonymizes all personal data
- Analytics and admin logs are anonymized after retention period

## Subject Access/Export

- Users can request export of all personal data
- Export includes all profile, progression, and reflection data

## Audit Log Retention and Access

- All admin and sensitive access is logged
- Logs retained for 2 years (default)
- Access to logs is restricted to authorized admins

## Telemetry Minimization

- Only collect analytics needed for product improvement
- Avoid storing unnecessary personal identifiers in events
